# RESTURANT-Pro-Admin-Panel
RestaurantPro Suite is a comprehensive and modern digital dashboard designed to be the central command center for restaurant management. It empowers restaurant owners and managers to streamline operations, monitor performance, and make data-driven decisions through a sleek, intuitive, and dark-themed interface.


## Run Locally

**Prerequisites:**  Node.js



# Clone the repository
git clone https://github.com/udip15/RESTURANT-Pro-Admin-Panel
.git

# Navigate into the folder
cd confession-site

# Install dependencies
npm install

# Start the development server
npm run dev
